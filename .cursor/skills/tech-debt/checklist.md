# 技术债务检查清单

## 代码层

- [ ] 重复代码 > 3 处相似逻辑
- [ ] 方法 > 50 行
- [ ] 类 > 500 行或职责 > 1
- [ ] 圈复杂度 > 10
- [ ] 嵌套层级 > 4
- [ ] 空 catch 或仅 log 不处理
- [ ] 魔法数字/字符串未提取常量

## 架构层

- [ ] 模块间循环依赖
- [ ] 接口与实现 1:1 无扩展点
- [ ] 全局单例滥用
- [ ] 缺少领域边界

## 测试层

- [ ] 核心业务无单元测试
- [ ] 测试依赖真实 DB/网络
- [ ] 测试可读性差、无 Given-When-Then
